## Common Mobile Problems and How to Fix Them

**Game won't load.** Check your connection first. If that's fine, try a hard refresh in Chrome or clear your cache. On iOS, disabling Private Browsing often fixes mysterious load failures.

**Screen loads but stays blank.** Usually a JavaScript problem. In Chrome on Android: Settings → Site Settings → JavaScript → make sure it's enabled.

**No sound.** Check Silent mode, then check whether your browser has permission to play audio. Some browsers disable audio by default as a battery-saving measure — check your browser's site permissions.

**Game freezes mid-spin.** Typically a brief connection drop. Don't close the game immediately — wait a moment. If it stays frozen, close the browser, reconnect, and reopen the casino. The spin result was recorded server-side when you pressed Spin, so your balance will be correct when you return.

**Touch controls feel laggy.** Either the browser is running heavy on memory or your screen protector has degraded. Try reloading the page. If that doesn't work, switch browsers.

**Pixelated or blurry graphics.** Almost always a browser zoom issue. Make sure you're at 100% zoom — pinch to reset if you've accidentally zoomed in.
