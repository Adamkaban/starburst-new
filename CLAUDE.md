## Page types — ALWAYS check before creating any page

TYPE 1 — Content pages (text from .md via [slug].astro)
  features, how-to-play, rtp-volatility, bonuses,
  strategy, mobile, casinos, faq, review
  Source: src/content/pages/[slug].md → rendered by src/pages/[slug].astro
  casinos.md: long-form text + casino table in markdown
  Promo code: stored as PROMO_CODE in src/data/casinos.ts

TYPE 2 — Special pages (dedicated .astro file, no .md source)
  demo.astro → lazy-load iframe + CTA, no long-form content

TYPE 3 — Unique pages (fully custom .astro)
  index.astro                  → Hero + StatGrid + ProsCons + CTA
  about.astro                  → Author bio, E-E-A-T signals
  responsible-gambling.astro   → NRGP info, helpline 0800 006 008
  privacy-policy.astro         → legal, noindex: true
  terms-of-use.astro           → legal, noindex: true

## Technical SEO — hard requirements (every page)
- PageSpeed: 90+ mobile AND desktop
- HTML tag: <html lang="en"> — English only, no hreflang needed
- hreflang: NOT required — single language (en), single geo (ZA)
- Canonical: always https://starburstslots.co.za/[slug]/ — https, non-www, trailing slash
- www/http→https: Cloudflare Pages handles https automatically
  www→non-www: set canonical domain in CF Dashboard (one-time setup)
- robots default: index, follow | exceptions: privacy-policy, terms-of-use → noindex
- Sitemap: all indexable pages included; noindex pages excluded
- H1: exactly ONE per page — no exceptions
- H-nesting: never skip levels (H1→H2→H3, not H1→H3)

## Schema.org — required on every page (JSON-LD in BaseLayout)
- WebSite + Organization: homepage only
- BreadcrumbList: every page (including homepage)
- Article + Author: all Type 1 content pages (including casinos)
- ItemList: /casinos/ (in addition to Article)
- Review: homepage
- FAQPage: /faq/ — via schema: "faq" in frontmatter
- Person (Author): /about/

## Meta tag limits
- Title: ≤44 chars WITHOUT spaces (≈55–65 chars with spaces)
  Pattern: "[Topic] – Starburst Slots ZA"
- Description: ≤220 chars | target 140–160 for quality
- All canonical URLs: https:// prefix, trailing slash

## Footer required pages (every page via BaseLayout)
- About | Privacy Policy | Responsible Gambling | Terms of Use

## Naming conventions
- Files/folders: kebab-case (how-to-play.astro, cta-box.astro)
- Components: PascalCase export (<CTABox />, <FAQSection />)
- CSS: Tailwind classes only — no BEM, no custom class names
- Variables/functions: camelCase | Constants: UPPER_SNAKE_CASE (AFFILIATE_URL)

## Hard constraints
- NO animations — no @keyframes, no CSS transition, no JS transitions
- Images: WebP only — width + height + alt always required
- loading="lazy" all images except above-fold hero (loading="eager" fetchpriority="high")
- Responsible gambling disclaimer: required on EVERY page (footer)
- Affiliate links: rel="nofollow noopener noreferrer" target="_blank" always
- Never hardcode affiliate URL — use AFFILIATE_URL from src/data/casinos.ts
- Never hardcode promo code — use PROMO_CODE from src/data/casinos.ts
